Ocultando información con Esteganografía

Esta es una pequeña demo que muestra la funcionalidad de la librería steganography.js. Básicamente permite ocultar o leer datos almacenados en una imagen.

Respecto a la demo: primero, puedes ocultar el texto del área de texto dentro de cualquier imagen que hayas seleccionado previamente de tu sistema de archivos local usando el botón "Ocultar". Una vez generada la nueva imagen con el texto codificado, puedes descargarla y distribuirla con el mensaje secreto en lugar de la antigua imagen sin mensaje. Como segunda parte, es posible leer un mensaje codificado dentro de la imagen seleccionada usando el botón "Leer".

Ten en cuenta que no habrá ningún mensaje si no se configura una imagen para que funcione el algoritmo, por lo que te recomiendo usar esta demo para codificar y decodificar el mensaje.
